# Code Node — Actividades Semanales

Repositorio de actividades y proyectos del bootcamp **[Code Node](https://codenode.franmoreno.com/)**, un programa de formación intensiva en desarrollo web y apps 100% online.

## Estructura del repositorio

Las actividades se organizan por semana. Cada semana contiene los ejercicios y proyectos propuestos por el programa.

```
code-node/
├── semana1/   # Introducción y Conceptos Básicos
├── semana2/   # Desarrollo con WordPress (Parte 1)
├── semana3/   # Desarrollo con WordPress (Parte 2)
├── semana4/   # Desarrollo con WordPress (Parte 3)
├── semana5/   # Desarrollo web con Astro (Parte 1)
├── semana6/   # Desarrollo con Astro (Parte 2)
├── semana7/   # Desarrollo con Next.js (Parte 1)
├── semana8/   # Desarrollo con Next.js (Parte 2)
├── semana9/   # Uso de IA en Desarrollo Web
└── semana10/  # Herramientas No-Code y Low-Code
```

## Temario

| Semana | Tema                                                   |
| ------ | ------------------------------------------------------ |
| 1      | HTML, CSS, JS, PHP básico, Git/GitHub, XAMPP           |
| 2      | WordPress: instalación, temas y plugins                |
| 3      | WordPress: temas desde cero, ACF, WooCommerce          |
| 4      | WordPress: plugins personalizados, hooks, REST API     |
| 5      | Astro: arquitectura, componentes, Tailwind, MDX        |
| 6      | Astro: datos dinámicos, APIs externas, SSG vs SSR      |
| 7      | Next.js: estructura, routing, fetching de datos        |
| 8      | Next.js: APIs, bases de datos, autenticación (Auth.js) |
| 9      | IA en desarrollo web: ChatGPT, Copilot                 |
| 10     | No-code / Low-code: Webflow, Shopify                   |
